# GloApp 

GloApp GitHub Repositories of Bangkit Academy 2023 Capstone Project Team C23-PS245 from 3 learning path, Machine Learning, Cloud Computing, and Mobile Development

## Features
- Registration
- Login
- News feed
- Profile
- Scan skin disease
- List of doctors
- Doctor's information
